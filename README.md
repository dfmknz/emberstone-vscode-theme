# emberstone-vscode-theme
A Visual Studio Code theme that goes with the Emberstone Omarchy theme.
=======
# Installation

1.  Choose **Extensions** from menu
2.  Search for `emberstone-theme-vscode`
3.  Click **Install** to install it
4.  Click **Reload** to reload the Code
5.  From the menu bar click: Code > Preferences > Color Theme > **Emberstone**
